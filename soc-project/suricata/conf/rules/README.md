# Emerging Threats Open Rules
# This file serves as a placeholder for ET Open Rules download
# Rules will be downloaded automatically when Suricata starts
# 
# Download latest rules from: https://rules.emergingthreats.net/
# 
# To download manually:
# wget https://rules.emergingthreats.net/open/suricata-6.0.8/emerging.rules.tar.gz
# tar -xzf emerging.rules.tar.gz -C /etc/suricata/rules/
#
# Main rule categories:
# - emerging-attack_response.rules: Attack response detection
# - emerging-botcc.rules: Botnet Command & Control
# - emerging-current_events.rules: Current threat events
# - emerging-dos.rules: Denial of Service
# - emerging-exploit.rules: Exploit attempts
# - emerging-malware.rules: Malware detection
# - emerging-scan.rules: Port scanning
# - emerging-web_client.rules: Web client attacks
# - emerging-web_server.rules: Web server attacks
# - tor.rules: Tor exit nodes
#
# For this lab, we use ET Open Rules which are free for non-commercial use
# Commercial use requires ET Pro subscription
